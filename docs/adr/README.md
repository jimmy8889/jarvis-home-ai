# Architecture Decision Records

Pilot OS uses Architecture Decision Records (ADRs) for material technical decisions.

## Status values

- Proposed
- Accepted
- Superseded
- Rejected

## Required sections

Each ADR must include:

1. Context
2. Decision
3. Consequences
4. Alternatives considered
5. Security and privacy impact
6. Operational impact
